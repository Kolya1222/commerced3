# Commerce Dashboard для Evolution CMS 3.x

Визуализирует продажи, поведение заказов и популярность товаров с помощью библиотеки **D3.js**.

## Возможности

- **Ключевые метрики (KPI)**: выручка сегодня, количество заказов, средний чек с динамикой относительно вчерашнего дня.
- **Линейный график выручки** с областью, сглаживанием и анимацией отрисовки.
- **Тепловая карта заказов** по дням недели и часам.
- **Воронка статусов** — распределение заказов по статусам (Новый → Завершён).
- **Treemap категорий** — площадь прямоугольника пропорциональна продажам.
- **Пузырьковая диаграмма (Circle Packing)** категорий.
- **Топ‑10 товаров** по количеству продаж (названия берутся из сохранённых заказов).
- **Sankey‑диаграмма переходов** заказов между статусами.
- Фильтр по периоду (7 дней, 30, 90, год) с динамической подгрузкой данных.
- Индикаторы загрузки (спиннеры) и анимированное появление графиков.

## Требования

- Evolution CMS ≥ 3.1.x
- Модуль [Commerce](https://github.com/evocms-community/commerce) установлен и настроен
- PHP 8.1

## Установка

Выполните команды из директории `/core`:

1. Установка пакета

```Shell
php artisan package:installrequire roilafx/commerced3 "*"
```

2. Публикация стилей и скриптов

```Shell
php artisan vendor:publish --provider="roilafx\commerced3\Commerced3ServiceProvider"
```
